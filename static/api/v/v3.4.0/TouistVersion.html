Module TouistVersion

module TouistVersion: sig .. end
Version information for the TouIST library.

This file is computed by cppo (see Makefile).

  • TouistVersion.version is the OPAM version (as seen in the _oasis file)
  • TouistVersion.has_git_tag tells if the library has been built inside a git repo; if it has, then TouistVersion.git_tag contains the latest git tag using git describe --tags

val version : string
val has_git_tag : bool
val git_tag : string
val has_yices2 : bool
val has_qbf : bool
