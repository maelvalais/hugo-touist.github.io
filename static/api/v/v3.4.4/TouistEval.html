Module TouistEval

module TouistEval: sig .. end

Evaluate an AST produced by TouistParse.parse_sat (or any other parse function) so it becomes a semantically correct formula.

eval is the main function.

val eval : ?smt:bool -> ?onlychecktypes:bool -> TouistTypes.Ast.t -> TouistTypes.Ast.t

eval checks the types, evaluates TouIST-specific constructs (variables, bigand, bigor, let...) and returns an evaluated formula.

ast is the AST given by TouistParse.parse_sat (or any other parsing function).

onlychecktypes will limit the evaluation to its minimum in order to get type errors as fast as possible.

smt enables the SMT mode. By default, the SAT/QBF mode is used.

  • Raises TouistErr.TouistFatal msg where TouistErr.msg contains the error.
val ast_without_loc : TouistTypes.Ast.t -> TouistTypes.Ast.t
val has_top_or_bot : TouistTypes.Ast.t -> bool
